## 🛠 주요 기능

![image](https://user-images.githubusercontent.com/67899069/144739988-50e1e88e-5f42-42c9-a4a3-e371b79ec90c.png)

### 1. 캐릭터 이동

- 월드 내에서 사용자는 캐릭터를 이동할 수 있습니다.
- 상대방 캐릭터의 위치를 실시간으로 확인 할 수 있습니다.

![image](https://user-images.githubusercontent.com/67899069/144739992-78bd995c-7814-428b-a5e8-7a91b23d25c5.png)

### 2. 채팅

- 월드 내 모든 사용자와 채팅을 주고받을 수 있습니다.
- 동일한 건물 내부의 사용자 끼리도 채팅을 주고받을 수 있습니다.
- 다른 건물의 사람들과는 전체 채팅만을 주고받을 수 있습니다.

![image](https://user-images.githubusercontent.com/67899069/144739996-b35711a2-a4fe-4b2d-870c-76dd06e59aab.png)

### 3. 화상 채팅

- 건물 내의 사용자끼리 비디오, 오디오 기반의 통신이 가능합니다.
- on, off 버튼을 통해 비디오, 오디오 제어가 가능합니다.

![image](https://user-images.githubusercontent.com/67899069/144740000-6b498d41-a07a-48ce-8ddc-91c4609b31bf.png)

### 4. 건물 생성

- 사용자는 월드에서 건물을 생성할 수 있습니다.
- 각 건물은 private, public의 접근성을 설정할 수 있습니다.
- private 건물은 패스워드를 입력해야 접근할 수 있습니다.

![image](https://user-images.githubusercontent.com/67899069/144740001-81570499-a65c-48a0-b015-4f6bc0cc609c.png)

### 5. 오브젝트 배치 및 파일 공유

- 월드 내부에 오브젝트를 빌드할 수 있습니다.
- 오브젝트에 파일을 첨부하여 여러 사람이 공유할 수 있습니다.
